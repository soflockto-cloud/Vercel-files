🔐 Ad Link Verification System

How It Works:

Step 1: User clicks "Complete Ad Link" button

Step 2: Ad opens in new window with unique verification token

Step 3: User completes the ad task

Step 4: Ad service redirects to ad-callback.html with the token

Step 5: Callback verifies token and signals completion

Step 6: Key automatically generates on main page

📝 Setup Instructions:

1. Configure Your Ad Service:

Tell your ad service provider to redirect to:

https://soflockto-cloud.github.io/Vercel-files/ad-callback.html?ref={VERIFICATION_TOKEN}

2. How It Verifies:

- Each user gets a unique token: verify_XXXXXX_timestamp

- Token is passed through ad service

- Ad service returns same token after completion

- Token match = verified completion

- Key is only generated on verified match

🛡️ Anti-Fraud Protection:

✓ Unique tokens prevent tampering

✓ Tokens expire after 10 minutes

✓ Only matching tokens generate keys

✓ Users can't skip by just switching tabs
