# Vercel-files

A GitHub Pages site that generates random keys upon completion of an ad link.

## Features
- Random key generation
- Ad link integration
- GitHub Pages deployment

## Getting Started

1. Clone this repository
2. Set up GitHub Pages in repository settings
3. Customize the ad link in `index.html`
4. Deploy to github.io

## Usage

Visit the deployed site and complete the ad link to receive a randomly generated key.
